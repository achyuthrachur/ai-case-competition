# Environment Setup Guide

Follow these steps before starting the competition.

## Step 1: Install VS Code
Download and install from: https://code.visualstudio.com/

VS Code is a free, lightweight code editor. You will use it to write and preview your HTML dashboard.

## Step 2: Open Your Project Folder
1. Open VS Code
2. Click **File > Open Folder**
3. Create a new folder called `meridian-competition` on your desktop
4. Open that folder in VS Code

## Step 3: Install the Live Server Extension
1. In VS Code, click the **Extensions** icon in the left sidebar (or press `Ctrl+Shift+X`)
2. Search for **Live Server**
3. Click **Install**

This lets you preview your HTML dashboard in the browser with live reloading.

## Step 4: Download the Dataset
Go to the **Downloads** tab on this site and download `transactions.csv` into your project folder.

## Step 5: Start Building with AI
Use AI tools to help you build your dashboard and write your memo:

- **ChatGPT** (https://chat.openai.com) — Ask it to help you write HTML, CSS, and JavaScript for your dashboard. Paste in sample rows from the dataset and ask it to generate charts, tables, or visualizations.
- **OpenAI Codex / ChatGPT with code** — Describe what you want your dashboard to do and ask it to write the code. You can paste error messages back into the chat for debugging.
- There are no rules on how much AI assistance you use — use it as much as possible.

## Step 6: Preview Your Dashboard
1. Right-click your `.html` file in VS Code
2. Select **Open with Live Server**
3. Your dashboard opens in the browser and updates automatically as you save

## Tips
- Keep your final dashboard as a **single `.html` file** with all CSS and JavaScript embedded inline — no external files
- Save your work frequently (`Ctrl+S`)
- When asking ChatGPT for help, include context: paste in the column names from the data dictionary and describe what anomaly patterns you want to visualize
